===== Requerimentos gerais para suporte ao LEDE =====
<WRAP 1000px>
<columns 100% 45% ->
  - SoC / [[docs:targets:|target]] suportado por LEDE
  - Flash necessário para acomodar a imagem do firmware do LEDE
    * 4MB no mínimo (não é possível instalar a interface web (LuCI))
    * 8MB é melhor (vai caber a interface web e mais outras aplicações)
  - RAM suficiente para uma operação estável
    * 32MB no mínimo, 64MB é melhor
